# 💫 About Me:
🌟 **Senior Full Stack Engineer · Infrastructure Architect · FinTech Platform Lead**

Hi there! 👋 I'm **Boopalan Pichandi** — a Senior Full Stack Engineer and Infrastructure Architect with **5+ years** building production-grade fintech platforms, cloud-native systems, and AI-driven financial analytics solutions.

🏦 **What I Build**
- End-to-end **FinTech investment intelligence platforms** — from React.js UI to cloud infrastructure
- **Real-time financial data pipelines** with Apache Kafka & Google Cloud Pub/Sub
- **AI-powered chatbots & LLM systems** for financial guidance and portfolio analytics
- **DevSecOps automation** — CI/CD pipelines, containerized microservices, zero-downtime deployments

☁️ **Infrastructure I Own**
- Cloud-native architecture on **AWS** (EC2, S3, RDS, Lambda) and **GCP**
- **Docker + Kubernetes** for horizontal scaling and production reliability
- **ELK Stack** for centralized observability and real-time dashboarding
- **PostgreSQL** with connection pooling on AWS RDS for large-scale financial datasets

🤖 **Currently Exploring**
- Advanced **LLM integrations** and AI agent workflows for financial analytics
- **Data Engineering** pipelines with Apache Beam and streaming architectures
- **Broker API integrations** (Zerodha/Kite) for algorithmic trading platforms

🤝 **Open To**
- Senior Engineering & Architecture roles in FinTech / Cloud / AI
- Collaborative AI innovations in the financial space
- Consulting on scalable platform architecture

💡 **Fun Fact**
When I'm not coding, I'm designing FinTech brand content in Figma and brainstorming ways to bridge DevOps, AI, and financial markets!
